CSS syntax
CSS syntax : โครงสร้างทางไวยากรณ์ และรูปแบบของภาษา CSS
CSS syntax ประกอบด้วยอะไรบ้าง : 1.seletor(ตำแหน่งที่เลือกจะกำหนดลักษณะ) 2.declaration(การประกาศลักษณะ)
รูปแบบโครงสร้าง css : selector { property:value } , selector { property1:value1; property2:value2 }


CSS
การเขียน CSS มีกี่แบบอะไรบ้าง
Answer: 3 แบบ 1.Inlune CSS: เขียนใน tag html
เช่น"< h1 style="color:blue;text-align:center;">This is a heading< /h1>"
2.external CSS: เขียนแยกไฟล์โดย link เข้ากับไฟล์ CSS จากภายนอก
เช่น < link rel="stylesheet" type="text/css" href="mystyle.css">
3.Internal CSS: เขียน < style> ไว้ภายในส่วนของ < head>ของหน้า HTML

Combinator
Combinator มีกี่แบบอะไรบ้าง
Answer : 6 แบบ 1.Comma Combinator "A,B"   
Ex.   a, p, div สามารถให้เราเลือก element ได้มากกว่า 2 ตัวเป็นการรวมตัวเลือก
2.Descendant selector (space) : a b
Ex.   เลือก Tag b ที่อยู่ภายใน Element a ทั้งหมด
3.Child selector(>) : a>b
Ex.   เลือก b ที่เป็น Child element ของ a
4.Adjacent sibling selector : a+b
Ex.   เลือก element b ที่ติดตาม a เช่น div + a เลือก < a> ที่อยู่หลัง < div>ในกรณี < div>มี < a>ตามหลังสองตัวเอาเพียงแค่ตัวเดียวคือตัวแรกที่ติด < div>
5.General sibling selector(~) : a~b
Ex.   มีลักษณะการใช้งานที่คล้าย a+b ต่างที่ a~b จะเลือก b ทั้งหมดหลัง a ถ้า a+b จะเลือก b แค่ตัวเดียวภายใน a
6.Universal Selector(*) : a*
Ex.   มีความหมายคือเลือกทุกอย่างเช่น p* เลือกทุก element ที่อยู่ภายในทุก < p> element
