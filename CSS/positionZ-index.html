Position คืออะไร และ มีอะไรบ้าง
Position เป็นเครื่องมือหนึ่งที่ใช้กำหนดตำแหน่ง element ต่างๆบน layout
1. static :เป็นค่าพื้นฐานของทุก block เป็นค่า default ของ position
2.relative :เป็นการจัดตำแหน่งจากจุดดั้งเดิมปกติสามารถกำหนดตำแหน่งที่จะไปจากจุดเดิมได้โดยจะยึดตำแหน่ง elementก่อนหน้า (top,bottom,left,right)
3.fixed :เลื่อนไปที่ไหนก็ได้ใน viewport มีความสัมพันธ์กับ viewport โดยจะพิเศษคือกล่องที่โดน fixed จะถูกตรึงไว้ตำแหน่งเดิม ไม่ว่าจะ scroll ไปตำแหน่งไหน มันก็จะอยู่ที่ตำแหน่งนั้น
4.absolute :จัดวางตำแหน่งโดยอิสระโดยยึดตาม parent element ที่ครอบ
5.sticky :จัดตำแหน่งตามตำแหน่งการเลื่อนของผู้ใช้ คล้ายกับการนำ relative กับ fixed ไว้ด้วยกัน relative กำหนดตำแหน่งเมื่อเราเลื่อนไปจนถึงตำแหน่งที่กำหนด จะทำการ fixed ทำให้ กล่องนั้นเคลื่อนที่ตามเราไปเหมือนใช้ fixed ต้องมีการระบุค่าตำแหน่งอย่างน้อย 1 ค่าเพื่อใช้งาน
Z-index คืออะไรใช้อย่างไร
Z-index คือ การจัด element ในรูปแบบชั้น layer
วิธีการใช้ Z-index จะคล้าย order คือเรียงตามค่าตัวเลข โดยปกติทุกตัวจะมีค่า Z-index=0 ถ้ากำหนดค่ามากจะแสดงอยู่ด้านหน้า แต่ถ้าเป็นค่า ลบ จะถูกซ้อนอยู่ข้างล่าง มองภายรวมเป็นชั้นๆ layer

position: static;

element with position: static; ไม่ได้อยู่ในตำแหน่งพิเศษใด ๆ มันจะถูกวางตำแหน่งตามการไหลปกติของหน้า
