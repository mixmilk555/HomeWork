Value ใน Display Property มีอะไรบ้าง
1.inline : ทำให้ element อยู่ในบรรทัดเดียวกันแต่ไม่สามารถปรับแต่ width และ height ได้
2.block : element แต่ละอันอยู่คนละบรรทัด
3.contents : ทำให้ elementนั้นกลายเป็น contents ไม่สามารถปรับ margin, padding, border, background ได้
4.flex : การจัดรูปแบบ flex
5.grid : การจัดรูปแบบ grid
6.inline-(block, flex, grid, table) : ทำให้ inline มีคุณสมบัติเพิ่มขึ้น
7.run-in : block or line ตามเนื้อหาของเรา


Grid และ Flex แตกต่างกันอย่างไร
1.ข้อแตกต่างแรก Grid : เป็นการจัด layout รูปแบบ two dimensions หรือ 2มิติ
Flex : เป็นการจัด layout รูปแบบ one dimensions หรือ 1มิติ
2.ข้อแตกต่างสอง Grid : ส่วนมากจะใช้ครอบ flex container ใช้ในเรื่องการจัดพื้นที่โดยรวมของ container และให้ flex จัดตำแหน่ง item ภายใน
Flex : ส่วนมากจะใช้จัดตำแหน่ง item ภายใน container เราสามารถนำ flex container อยู่ภายใน grid container ได้แต่ใช้สลับกันไม่ได้


Justify and Align แตกต่างกันอย่างไร
ข้อแตกต่าง Justify : เป็นการจัด item ที่อยู่ใน container แนวนอน ตามแกน y แต่ใน flex ถ้ามีการกำหนดแสดงผลแบบ column justify จะเป็นการจัดแบบแนวตั้ง เพราะเป็นการสลับแกน
align : เป็นการจัด item ที่อยู่ใน container แนวตั้ง ตามแกน x
สรุป = justify จัดเรียงในแนวนอน ส่วน align จัดเรียงในแนวตั้ง โดยปกติ คำสั่ง justify and align จะอยู่ในส่วนของ container เว้นแต่ align self ของ flexbox ที่คำสั่งจะอยู่ใน item box
