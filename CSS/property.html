Property และ Value คืออะไรแตกต่างกันอย่างไร
Property คือ คุณสมบัติ ลักษณะที่เรากำหนดให้เช่น width height color size เหมือนกับเราสร้างของเรากำหนดให้มีคุณสมบติอะไรบ้าง กว้าง ยาว สี สูง ให้แสดงผลตามนั้น
Value คือ ค่าของ Preperty ว่าเราจะกำหนดค่าให้อย่างไร เช่น สูงกี่ m สีอะไร ขนาดใหญ่เท่าไหร่ มีกรอบแล้วกรอบสีอะไร หนาเท่าไหร่ เป็นต้น เป็นค่าให้เราใส่ให้กับ property แสดงผลตามค่านั้น


Property ที่สอนใน precamp มีกี่กลุ่มอะไรบ้าง
Answer = 5 กลุ่ม 1.Property for content: color , size , font , text, list-style-type, Height, Width, background
2.Property for border:border, table
3.Property for padding:padding (top right buttom left)
4.Property for margin:margin (top right buttom left)
5.property for element:Display, Height, Width, Position, Float, Align, Flex, Grid


Value unit มีกี่ประเภทอะไรบ้างแต่ละ ประเภทคืออะไร
Absolute lenghts Relative lenghts
cm = เซนติเมตร em = โดย 1 em = 1เท่า ของ font-size เดิมของ parent element
mm = มิลลิเมตร ex = 1/2 em คล้ายกับ em ต่างที่ใช้ font-size ของ font ปัจจุบันที่ใช้ใน element
in = นิ้ว = 96 px = 2.54 cm ch = เพิ่มขนาดจากความการ * ความกว้างของ 0
px = pixels rem = เหมือน em แต่ต่างที่ rem จะยึดตาม root element ไม่ตามสืบทอดจาก parent element แม้ว่าจะอยู่ใน parent ก็ตาม โดยปกติขนาด font browser จะกำหนดไว้คือ 16 px เปลี่ยนโดยพิม html { font-size: Npx}
pt = points(1pt=1/72 of in) vw = 1% ของความกว้าง viewport ปรับขนาดตามขนาดหน้าต่างที่เราใช้
pc = picas(1pc=12pt) vh = 1% ของความสูง viewport
Vmax , Vmin = 1/100th ของขนาด viewport ที่ใหญ่กว่า และ เล็กกว่า ตามลำดับ
% = ยึดตาม parent element ปัจจุบัน
